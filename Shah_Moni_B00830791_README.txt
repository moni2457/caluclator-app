TIP CALC

The "TIP CALC" App is a modern app for Android platform written in Kotlin. It provides the functionality to calculate the tip as well as bill splitting. 
Your target audience is English speaking Haligonians, age 18 and above, going to restaurants, pubs, and diners. They will be using Canadian Dollars, 
are any sex/gender, and use Android phones less than 3 years old.

Features:
The app has a basic functionality to calculate the tip (0-25%) when provided with the bill amount and tip.

It also has an additional functionality of splitting the bill amount based on the number of people. Enter the bill amount, tip% and the number of persons.
This would result in the total bill, total tip, total bill, total bill per person and total tip per person.


Getting Started:
Your mobile device should be running on Android OS with Lollipop version (5.0 v) or above. There is no need for any network connection to use the application.
There is no additional requirements for app installation. It can be used easily installed and used on any device having the above requirements.
